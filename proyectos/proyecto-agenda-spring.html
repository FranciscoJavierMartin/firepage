Agenda de contactos

Se trata de un proyecto que recrea una agenda de contactos con las funcionalidades típicas tales como guardar contactos, ver los ya existentes, modificarlos o borrarlos de la base de datos. Para desarrollar todo el proyecto se han usado las siguientes tecnologías:

  • Spring

    Framework para el desarrollo web basado en la plataforma Java Enterprise.

  • Spring Boot

    Framework sobre el framework que simplifica la creación de proyectos. Dispone ademas de una herramienta web para crear proyectos con todas las dependencias necesarias.

  • Thymeleaf

    Motor de plantillas, el cual se integra perfectamente con Spring.

  • MySQL

    Motor de base de datos relacional open source.

  • Hibernate

    ORM (Object Relational Mapping) escrito en Java. Transforma las filas de una tabla de la base de datos

  • Spring Security

    Módulo del proyecto Spring para labores de seguridad. Se trata de uno de sus primeros módulos y uno de los mas desarrollados del proyecto.

  • Spring MVC

    Módulo basado en el patrón de arquitectura Modelo - Vista - Controlador. Separa la funcionalidad en capas para permitir una mejor modularidad del código.
